Panduan Setup G Suite Temp Email

Langkah demi langkah dari nol hingga sistem siap digunakan untuk *generate* unlimited akun.

LANGKAH 1

Mendaftar Domain Gratis

Untuk membuat Google Workspace, Anda membutuhkan sebuah nama domain. Jika Anda tidak ingin membeli domain berbayar (.com / .id), Anda bisa menggunakan layanan penyedia subdomain gratis yang mendukung pengaturan DNS (TXT Record).

Cara Daftar di DNSExit.com (Sangat Direkomendasikan):
1. Buka website dnsexit.com dan buat akun.
2. Masuk ke menu Domains -> Free Subdomains.
3. Ketik nama yang Anda inginkan (misal: kreaverse-app) dan pilih akhiran yang tersedia (misal: .2bd.net).
4. Selesaikan proses *checkout* ($0.00). Domain Anda langsung aktif!

3 Rekomendasi Website Domain Gratis (Terbukti Berhasil 100%):

DNSExit.comFree Subdomains (.2bd.net dll)
FreeDomain.oneSatu server dengan DNSExit
Dynu.comFree DDNS (.freeddns.org dll)
LANGKAH 2

Daftar Cloud Identity Free & Verifikasi

Setelah memiliki domain, langkah selanjutnya adalah mendaftarkannya ke Google untuk mendapatkan kapasitas 50 akun gratis.

  1. Buka halaman pendaftaran Google Cloud Identity Free (Klik di sini).
  2. Masukkan nama domain yang baru Anda buat di Langkah 1 (misal: kreaverse-app.2bd.net).
  3. Isi data diri dan buat akun Admin pertama Anda (misal: admin@kreaverse-app.2bd.net). Catat password-nya!
  4. Setelah berhasil login ke admin.google.com, Google akan meminta Anda Memverifikasi Kepemilikan Domain.
  5. Google akan memberikan sebuah kode TXT Record.
    Contoh bentuk kode dari Google:
    google-site-verification=90zsOND6L-2jdUeKaWTCMomNNeNmWVn_Dd96QpDxz2g
  6. Buka panel DNS di tempat Anda mendaftar domain (misal: DNSExit).
  7. Tambahkan TXT Record baru. Kosongkan kolom Host/Name (atau isi dengan @), lalu *paste* kode panjang dari Google tersebut ke kolom Value / Teks. Simpan.
  8. Kembali ke Google Admin dan klik Verifikasi. (Tunggu 5-15 menit jika gagal).
LANGKAH 3

Mendapatkan Kunci JSON (Jalan Ninja Anti-Error)

Untuk menghubungkan Google Admin ke Dashboard Kreaverse, kita butuh file JSON. Seringkali pemula mengalami error saat mencoba membuat Kunci JSON di akun Workspace baru.

SOLUSI ERROR "Service account key creation is disabled" (JALAN NINJA):
Jika Anda mendapat error merah karena Organization Policies saat membuat Kunci JSON, Jangan buat Project menggunakan akun Workspace Anda!

Gunakan akun @gmail.com pribadi Anda untuk membuat Kunci JSON. Kunci dari Gmail pribadi tidak terkena blokir kebijakan dan tetap bisa dipakai untuk membuka Workspace manapun!
  1. Buka Google Cloud Console dan login menggunakan akun @gmail.com pribadi Anda.
  2. Buat Project Baru (Bebas beri nama apa saja).
  3. Di kolom pencarian atas, ketik Admin SDK API, lalu klik Enable (Aktifkan).
  4. Klik menu garis tiga di kiri atas -> IAM & Admin -> Service Accounts.
  5. Klik + Create Service Account dan isi nama bebas.
Tanya Jawab Penting Saat Membuat Kunci:
  • Pilih Role (Peran) apa di Step 2?
    KOSONGKAN SAJA (Lewati). Anda tidak perlu memilih peran apapun. Langsung klik Continue / Done.
  • Di mana letak "OAuth 2 Client ID"?
    Angka ini ada di tabel daftar Service Account. Atau jika Anda mengklik nama Service Account-nya, angkanya ada di bagian paling atas (tepat di bawah kotak pencarian).
  1. Copy (Salin) angka OAuth 2 Client ID tersebut dan simpan di Notepad.
  2. Klik titik tiga di sebelah kanan Service Account tersebut -> Manage Keys.
  3. Klik Add Key -> Create new key -> Pilih JSON -> Klik Create. File JSON akan terdownload ke HP/Laptop Anda.
LANGKAH 4

Memberikan Izin Delegasi Domain

Kunci JSON sudah ada, sekarang kita harus menyuruh Google Admin untuk "menerima" kunci tersebut agar bisa membuat dan menghapus email.

  1. Login ke admin.google.com menggunakan akun admin Workspace Anda (misal: admin@kreaverse-app.2bd.net).
  2. Di menu kiri, masuk ke Keamanan -> Kontrol akses dan data -> Kontrol API.
  3. Scroll ke bawah, klik Kelola Delegasi Beri Akses Domain.
  4. Klik Tambahkan Baru.
  5. Di kolom Client ID, *paste* angka yang Anda simpan di Notepad pada Langkah 3.
  6. Di kolom Cakupan OAuth, *copy-paste* kode di bawah ini:
https://www.googleapis.com/auth/admin.directory.user
  1. Klik Otorisasi. Selesai! Google Admin Anda sekarang sudah terbuka untuk Kreaverse.
LANGKAH 5

Hubungkan ke Dashboard Kreaverse

Langkah terakhir! Mari kita masukkan Kunci JSON tadi ke dalam Dashboard Admin Kreaverse Anda.

  1. Buka Dashboard Admin Kreaverse AI Anda.
  2. Masuk ke menu G Suite Temp Email.
  3. Di panel Penyetelan Google Workspace API, klik tombol hijau + Tambah Akun Admin.
  4. Masukkan email admin Workspace Anda (misal: admin@kreaverse-app.2bd.net).
  5. Buka file JSON yang Anda download di Langkah 3 menggunakan aplikasi teks (Notepad/Teks Editor). Copy seluruh isi teksnya dari kurung kurawal awal { sampai akhir }.
  6. *Paste* seluruh teks JSON tersebut ke kolom yang disediakan di Dashboard.
  7. Klik Simpan.
SELAMAT! SISTEM SUDAH SIAP!
Sekarang Anda bisa langsung menggunakan fitur Auto Generate (Gen Z) untuk membuat puluhan akun G Suite secara otomatis. Sistem akan membagi beban (pooling) maksimal 30 akun per organisasi. Jika ingin lebih, cukup ulangi Langkah 1-5 untuk domain baru!
